## 📌 Fitur

- 🔍 **Market Scanner** — Pantau harga & volume semua pasar aktif
- 🐋 **Whale Tracker** — Deteksi pergerakan besar (≥ $10.000)
- 💰 **Arbitrage Signal** — Cari selisih harga antar market terkait
- 🚨 **Price Alert** — Notifikasi saat harga mencapai level tertentu
- 📱 **Telegram Notifier** — Kirim alert ke Telegram pribadi/group
- 💬 **Discord Notifier** — Kirim alert ke Discord via webhook
- ⚡ **Zero Dependency** — Hanya pakai Python stdlib, tanpa install tambahan
